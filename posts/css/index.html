字体样式

属性 说明
font-family 字体类型
font-size 字体大小
font-weight 字体粗细
font-style 字体风格
color 字体颜色

字体类型：font-family

默认为宋体 设置语法

<style type="text/css"> 
#div1{font-family: Arial;}
        #div2{font-family: "Times New Roman";}
        #div3{font-family: "微软雅黑";}
    </style>

例子：

<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8" />
    <title></title>
    <style type="text/css">
        #div1{font-family: Arial;}
        #div2{font-family: "Times New Roman";}
        #div3{font-family: "微软雅黑";}
    </style>
</head>
<body>
    <div id="div1">Arial</div>
    <div id="div2">Times New Roman</div>
    <div id="div3">微软雅黑</div>
</body>
</html>

一个实践：

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>: )</title>
    <style type="text/css">
        .mycolor{color: blue}
        #font1{font-family: mononoki;}
        #font2{font-family: "Times New Roman";}
    </style>
</head>
<body>
    <div class="mycolor">
    <div id="font">mononoki</div>
    </div>
</body>
</html>

对于font-family属性，如果字体类型只有一个英文单词，则不需要加上双引号；如果字体类型是多个英文单词或者中文的，则需要加上双引号。注意，这里的双引号是英文双引号，而不是中文双引号。

字体大小：font-size

通常采用px方式设置而不是关键字

设置方式：

<style type="text/css">
        #p1 {font-size: 10px;}
        #p2 {font-size: 15px;}
        #p3 {font-size: 20px;}
    </style>

一个例子：

字体粗细：font-weight

取值方式有关键字和100-900的数值，实际开发通常只会使用到bold

设置方式

    <style type="text/css">
        #p1{font-weight:lighter;}
        #p2{font-weight:normal;}
        #p3{font-weight:bold;}
        #p4{font-weight:bolder;}

字体风格：font-style

属性 说明
normal 正常

设置方式

    <style type="text/css">
        #p1{font-style:normal;}

字体颜色：color

取值方式：

  • 关键字
    <style type="text/css">

  • 16进制RGB值

        #p1{color: #03FCA1;}

简介

语法（同样对图片有效）

        #p1{text-align:left;}

语法：

    <style type="text/css">
        #p1{text-decoration:underline;}
        #p2{text-decoration:line-through;}

属性值 说明

line-through 中划线
overline 顶划线
  • html：结构控制
  • css：外观控制

    <style type="text/css">
        #p1{text-transform:uppercase;}
        #p2{text-transform:lowercase;}
        #p3{text-transform:capitalize;}

属性值 说明
none 无转换（默认值）
uppercase 转换为大写
lowercase 转换为小写
capitalize 只将每个英文单词首字母转换为大写

    <style type="text/css">
        #p1{line-height:15px;}
        #p2{line-height:20px;}

    </style>

</head>

Chapter5-边框样式

简介

属性 说明
border-width 边框的宽度
border-style 边框的外观
border-color 边框的颜色

想要为一个元素定义边框样式，必须要同时设置border-width、border-style、border-color属性才会有效果

整体样式

  • border-width

定义边框宽度，值是一个像素值 px

  • border-style

定义边框的外观常用取值如下

属性 取值
none 无样式
dashed 虚线
solid 实线
  • border-color

定义边框颜色，取值为16进制RGB

一个例子：

<style type="text/css">
        /*定义所有div样式*/
        div
        {
            width:100px;
            height:30px;
        }
        /*定义单独div样式*/
        #div1
        {
            border-width:1px;
            border-style:dashed;
            border-color:red;
        }
        #div2
        {
            border-width:1px;
            border-style:solid;
            border-color:red;
        }
    </style>

可简写为

    <style type="text/css">
        div{ border: 1px solid red; }
    </style>

局部样式

设置局部样式

	<style>{
            width: 100px;                       /*div元素宽为100px*/
            height: 60px;                       /*div元素高为60px*/
            border-top: 2px solid red;          /*上边框样式*/
            border-right: 2px solid yellow;     /*右边框样式*/
            border-bottom: 2px solid blue;      /*下边框样式*/
            border-left: 2px solid green;       /*左边框样式*/
         }
    </style>

border-bottom:0px;、border-bottom:0;和border-bottom:none;是等价的

本章练习

1．如果我们想要定义某一个元素的右边框，宽度为1px，外观为实线，颜色为红色，正确写法为（ C ）。

    A. border:1px solid red;
    B. border:1px dashed red;
    C. border-right:1px solid red;
    D. border-right:1px dashed red;

2．如果我们想要去掉某一个元素的上边框，下面写法中不正确的是（ A ）。

    A. border-top:not;
    B. border-top:none;
    C. border-top:0;
    D. border-top:0px;

Chapter7-表格样式

表格标题位置：caption-side

  • top:顶部
  • bottom：底部
    <style type="text/css">
        table,tr, td,th{border:1px solid silver;}
        table{caption-side:bottom;}
    </style>

表格边框合并：border-collapse

    <style type="text/css">
        table,th,td{border:1px solid silver;}
        table{border-collapse: collapse;}
    </style>

表格边框间距：border-spacing

    <style type="text/css">
        table,th,td{border:1px solid silver;}
        table{border-spacing: 8px;}
    </style>

本章练习

1．CSS可以使用（ C ）属性来合并表格边框

    A. border-width                   
    B. border-style
    C. border-collapse                  
    D. border-spacing

2．下面有关表格样式，说法不正确的是（ A ）。

    A. border-collapse只限用于表格，不能用于其他元素
    B. caption-side、border-collapse、border-spacing一般在table元素中设置
    C. 可以使用border-collapse: separate;来将表格边框合并
    D. 如果要为表格添加边框，我们一般需要同时对table、th、td这几个进行设置

Chapter8-图片样式

简介

    <style type="text/css">
        img
        {
            width: 60px;
            height: 60px;
        }
    </style>

在实际开发中，如果你需要多大的图片，就用Photoshop制作多大的图片。不建议使用一张大图片，然后再借助width和height来改变大小。因为一张大图片体积更大，会使页面加载速度变慢。这是性能优化方面的考虑，以后我们会慢慢接触

图片边框

    <style type="text/css">
        img
        {
            widht: 60px;
            height: 60px;
            border: 1px solid red;
        }
    </style>

图片对齐

水平对齐：text-align

text-align属性一般只用于两个地方：文本水平对齐和图片水平对齐。

属性值 说明
left 左对齐（默认值）
center 居中对齐
right 右对齐

例：

    <style type="text/css">
        div
        {
            width: 300px;
            height: 80px;
            border: 1px solid silver;
        }
        .div1{ text-align: left; }
        .div2{ text-align: center; }
        .div3{ text-align: right; }
         img{ width: 60px; height: 60px; }
    </style>

垂直对齐

属性 取值
top 顶部对齐
middle 中部对齐
baseline 基线对齐
bottom 底部对齐 
    <style type="text/css">
        img{ width: 60px; height: 60px; }
        #img1{ vertical-align: top; }
        #img2{ vertical-align: middle; }
        #img3{ vertical-align: bottom; }
        #img4{ vertical-align: baseline; }
    </style>
