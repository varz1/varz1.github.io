Posted at — Apr 6, 2020

定义

var 数组变量名 [元素数量]T

初始化

  • 默认值为0
  • [...]推测数组大小
  • 指定位置初始化某位置的值

遍历

  1. for
   cityArray := [...]string{"西安","莆田","上海"}
   	for i:= 0;i<3;i++{
   		fmt.Println(cityArray[i])
   	}
  1. for range
   for _,value:=range cityArray{
   		fmt.Println(value)
   	}

数组是值类型

数组是值类型，赋值和传参会复制整个数组。因此改变副本的值，不会改变本身的值。

注意：

  1. 数组支持 “==“、”!=” 操作符，因为内存总是被初始化过的。
  2. [n]*T表示指针数组，*[n]T表示数组指针 。

练习题

  1. 求数组[1, 3, 5, 7, 8]所有元素的和
//求数组`[1, 3, 5, 7, 8]`所有元素的和

package main

import "fmt"

func main() {
	Array := [...]int{1, 3, 5, 7, 8}
	value := 0
	for i := 0; i < 5; i++ {
		value += Array[i]
	}
	fmt.Println(value)
}
  1. 找出数组中和为指定值的两个元素的下标，比如从数组[1, 3, 5, 7, 8]中找出和为8的两个元素的下标分别为(0,3)和(1,2)。
//找出数组中和为指定值的两个元素的下标，比如从数组`[1, 3, 5, 7, 8]`中找出和为8的两个元素的下标分别为`(0,3)`和`(1,2)`。
package main

import "fmt"

func main() {
	var targetArray = [5]int{1, 3, 5, 7, 8}
	k := 8
	var m, n = Find(targetArray, k)
	fmt.Println(m, n)
}

func Find(arr [5]int, target int) (int, int) {
	var x, y ,i,j int
	x = arr[0]
	for i = 0; i < len(arr); i++ {
		for j = 0; j < len(arr); j++ {
			if arr[i]+arr[j] == target {
				goto tag
			}
		}
	}
	tag:
		x = i
		y = j
	return x, y
}
