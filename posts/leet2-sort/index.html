1.数组中的第K个最大元素

  1. Kth Largest Element in an Array (Medium)

Leetcode / 力扣

Input: [3,2,1,5,6,4] and k = 2
Output: 5

题目描述：找到倒数第 k 个的元素。

Java sort包

    public int findKthLargest(int[] nums, int k) {
        Arrays.sort(nums);
        return nums[nums.length - k];
    }
    //时间复杂度：O(NlogN) sort快排
    //空间复杂度：O(1)

堆排序

    public int findKthLargest(int[] nums, int k) {
        PriorityQueue<Integer> pq = new PriorityQueue<>();
        for (int val : nums) {
            pq.add(val);
            if (pq.size() > k)  // 维护堆的大小为 K
                pq.poll();
        }
        return pq.peek();
    }
//时间复杂度 O(NlogK)，空间复杂度 O(K)。

快速排序

public int findKthLargest3(int[] nums, int k) {
        int left = 0, right = nums.length - 1;
        k = nums.length - k;
        while (left < right) {
            int j = partition(nums, left, right);
            if (j == k)
                break;
            else if (j < k) {
                left = j + 1;
            } else {
                right = j - 1;
            }
        }
        return nums[k];
    }

    public int partition(int num[], int l, int h) {
        int i = l, j = h + 1;
        while (true) {
            while (num[++i] < num[l] && i < h) ;
            while (num[--j] > num[l] && j > l) ;
            if (i >= j) {
                break;
            }
            swap(num, i, j);
        }
        swap(num, l, j);
        return j;
    }

    public void swap(int[] a, int i, int j) {
        int t = a[i];
        a[i] = a[j];
        a[j] = t;
    }
//时间复杂度 O(N)，空间复杂度 O(1)

2.前 K 个高频元素

  1. Top K Frequent Elements (Medium)

Leetcode / 力扣

Given [1,1,1,2,2,3] and k = 2, return [1,2].

桶排序

    public List<Integer> topKFrequent(int[] nums, int k) {
        List<Integer> res = new ArrayList();
        // 使用字典，统计每个元素出现的次数，元素为键，元素出现的次数为值
        HashMap<Integer, Integer> map = new HashMap();
        for (int num : nums) {
            if (map.containsKey(num)) {
                map.put(num, map.get(num) + 1);
            } else {
                map.put(num, 1);
            }
        }

        //桶排序
        //将频率作为数组下标，对于出现频率不同的数字集合，存入对应的数组下标
        List<Integer>[] list = new List[nums.length + 1];
        for (int key : map.keySet()) {
            // 获取出现的次数作为下标
            int i = map.get(key);
            if (list[i] == null) {
                list[i] = new ArrayList();
            }
            list[i].add(key);
        }

        // 倒序遍历数组获取出现顺序从大到小的排列
        for (int i = list.length - 1; i >= 0 && res.size() < k; i--) {
            if (list[i] == null) continue;
            res.addAll(list[i]);
        }
        return res;
    }

3. 按照字符出现次数对字符串排序

  1. Sort Characters By Frequency (Medium)

Leetcode / 力扣

Input:
"tree"

Output:
"eert"

Explanation:
'e' appears twice while 'r' and 't' both appear once.
So 'e' must appear before both 'r' and 't'. Therefore "eetr" is also a valid answer.

桶排序

class Solution {
    public String frequencySort(String s) {
        HashMap<Character, Integer> hashMap = new HashMap<>();
        char[] str = s.toCharArray();
        for (char c : str) {
            //System.out.print(c + " ");
            if (hashMap.containsKey(c)) {
                hashMap.put(c, hashMap.get(c) + 1);
            } else {
                hashMap.put(c, 1);
            }//hashMap统计字符出现个数
        }//Key为字符Value为字符
        List<Character>[] bucket = new List[str.length + 1];
        for (Character character : hashMap.keySet()) {
            int index = hashMap.get(character);
//            System.out.print("key--"+character+" ");
//            System.out.println("value--"+index+" ");
            if (bucket[index] == null) {
                bucket[index] = new ArrayList<Character>();
            }
            bucket[index].add(character);
            //System.out.println(bucket[1]);
        }
        //创建List按顺序存储字符
        StringBuilder st = new StringBuilder();
        for (int i = str.length; i >= 0; i--) {
            if (bucket[i] == null) continue;
            for (Character character : bucket[i]) {
                //这边注意字符按出现次数append
                for (int j = 0; j < i; j++) {
                    st.append(character);
                }
            }
        }
        return new String(st);
    }
}

基础复习

1.StringBuilder类：Java StringBuffer 和 StringBuilder 类

  • 当对字符串进行修改的时候，需要使用 StringBuffer 和 StringBuilder 类。

和 String 类不同的是，StringBuffer 和 StringBuilder 类的对象能够被多次的修改，并且不产生新的未使用对象。

在使用 StringBuffer 类时，每次都会对 StringBuffer 对象本身进行操作，而不是生成新的对象，所以如果需要对字符串进行修改推荐使用 StringBuffer。

StringBuilder 类在 Java 5 中被提出，它和 StringBuffer 之间的最大不同在于 StringBuilder 的方法不是线程安全的（不能同步访问）。

由于 StringBuilder 相较于 StringBuffer 有速度优势，所以多数情况下建议使用 StringBuilder 类。

2.对于char类型数组的append方法

  • st.append(c)//添加元素
  • s.toCharArray()//字符串转换为字符数组

3.HashMap的方法

  • hashMap.containsKey()//是否包含某key
  • hashMap.put(key,value)//添加键值对
  • hashMap.keySet()//返回一个所有Key组成的Set
  • hashMap.get(c)//输入key返回对应value
