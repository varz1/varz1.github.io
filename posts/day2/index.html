iota

常量计数器，只能在常量的表达式中使用。

iota在const关键字出现时重置为0.const中每新增一行常量声明将使iota计数一次。iota可简化定义，在枚举时很有用

const (
	a1 = iota //0
	a2 = iota //1 
)

四个例子

1.

const(
	c1 = iota
	c2

	c3

2.
