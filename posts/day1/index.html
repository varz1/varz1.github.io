变量声明

  • 单个声明

    var 变量名 数据类型

  • 批量声明

var (
    a string
    b bool
    c int
)

  • 除非全局变量所有变量声明需使用

  • 三种Print

fmt.Print //无格式化
fmt.Printf //格式化输出 eg.fmt.Printf("name:%s",name)
fmt.Println //自动换行
  • 驼峰命名
var studentName string
  • 类型推导
var s2 = "halo"
  • 短变量声明（函数内部）
s1 := 1
  • 匿名变量

多重赋值中忽略某个值_

func foo()(int , string){
	return 2,"halo"
}
x,_ := foo()
	fmt.Println("x=",x)
